The most important requirement for changing status in Debian is to make sure that a person actually intends to, so this page serves to enter a signed statement of intent that can be posted to debian-newmaint.

{% if edit %}

This page allows to manage the statements of intent automatically. To add or change the signed agreement to SC/DFSG/DMUP, paste it in the form below.

An example statement of intent can be something like this: 

For nm.debian.org, at {{now|date:"Y-m-d"}}:
I would like to apply to change my status in Debian to {{process.applying_for|desc_status}}.
I have worked on X and Y for a while and I would like to be able to Z.
