Debian New Member - Join the NM process

We apologise this is not yet ready on the new web interface.

For now, to join the process please mail nm@debian.org taking care to:

  • Say that you want to become a Debian Developer
  • Say if you also need to upload packages
  • Cc your advocate(s)
  • Say what username you would like (minimum 3 characters, check db.debian.org to see if it is free)
  • Add anything else you may think is useful
  • Sign the email with the key you use to sign packages
